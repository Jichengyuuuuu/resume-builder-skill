> **不用反复打磨措辞，不用花钱找人优化，不用花大量时间整理材料。**  
> **把你有的东西——旧简历、目标岗位JD、随手写的工作流水账、工作项目文档——统统丢给模型，剩下的交给这个 Skill。**

## ✨ 为什么选择这个简历生成 Skill？

传统简历优化的问题：找人改，贵且慢；自己写，来回打磨措辞耗时耗力；用 AI，又总是需要反复"调教"才能得到满意的结果。

**Resume Builder Skill 的逻辑是：你负责提供素材，它负责转化成专业简历。**

- 🎯 **极简输入** — 旧简历、JD、工作流水账、项目文档，任何格式统统可以，一次输入直接成型
- 🧠 **深度定制，拒绝千篇一律** — 内置 11+ 岗位类型技能维度逻辑，后端工程师、产品经理、市场营销各有各的写法
- 🔍 **ATS 友好** — 关键词自动对齐目标岗位 JD，帮你的简历轻松通过企业招聘系统自动筛选
- 🗂️ **双格式输出** — 同时生成 HTML（在线预览）和 DOCX（Word 可编辑），无水印，完全归属于你
- 💡 **附赠"高阶顾问"求职建议** — 简历生成后自动输出 3-5 条专家视角的职场建议，助你提升求职胜率

## 🚀 快速开始

将 `resume-builder-skill.md` 的内容作为 System Prompt 或 Skill 注入你使用的 Agent / LLM，然后直接说：

```
帮我生成一份简历，我要投产品经理岗位，我的工作主要是……
```

或者把你有的任何材料（旧简历、JD、工作经历描述）作为附件一起发给模型：

```
帮我基于这份旧简历生成一份针对产品经理岗位的新简历（附上现有简历内容）
```

Skill 会自动完成信息收集与简历生成，一次性输出可投递的专业简历。

## 🔄 简历生成工作流程

```
用户提供背景信息（旧简历 / JD / 口头描述皆可）
                    ↓
             信息收集阶段
     · 个人基本信息 & 求职方向
     · 教育背景
     · 工作经历 + 量化成就
     · 岗位定制技能（按岗位类型动态调整）
     · 项目经验（可选）
                    ↓
             简历生成阶段
     · HTML 简历（支持在线预览 + 打印为 PDF）
     · DOCX 简历（Word / WPS 可直接编辑投递）
                    ↓
          高阶求职顾问建议
     · 3-5 条精准求职 & 面试建议
     · 面试高频考点预判
     · 个人亮点提炼
```

## 📋 输出示例

### 简历工作经历分点格式

每条工作描述采用 **"加粗概括 + 冒号 + 详细描述"** 格式，让面试官 5 秒内抓住重点：

```
• 系统架构设计：主导设计高并发分布式支付系统，日均处理交易 100 万+，可用性 99.99%
• 性能优化：优化核心查询链路，平均响应时间从 500ms 降至 50ms，提升 10 倍
• 团队管理：带领 5 人小组完成 Q3 季度目标，提前 2 周交付
```

### 高阶求职顾问建议示例

```
💡 高阶顾问建议（以资深产品总监视角）

1. 补充商业化认知：你的经历集中在效率工具侧，但目标岗位是「商业化产品经理」，
   建议补充你对营收指标、定价策略的理解
2. 亮点提炼 — 变革推动力：从纸质到线上不只是产品设计，更是组织变革故事，
   可在面试中重点强调推动阻力和落地策略
3. 面试高频考点：B 端商业化岗大概率会问「如何衡量产品商业价值」，
   建议用 ROI 框架提前准备
```
