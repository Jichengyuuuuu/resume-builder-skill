# 📄 Resume Builder Skill

> **不用反复打磨措辞，不用花钱找人优化，不用花大量时间整理材料。**  
> **把你有的东西——旧简历、目标岗位JD、随手写的工作流水账、工作项目文档——统统丢给模型，剩下的交给这个 Skill。**  
>
> Drop anything you have to any agent or LLM. No polishing, no expensive consultants, no prep work.

---

## ✨ 为什么选择这个 Skill？

大多数 AI 简历工具的问题：要么需要反复调整、来回对话，要么输出千篇一律的通用模板。

**Resume Builder Skill 不一样：**

- 🎯 **一次输入，直接成型** — 只需提供背景信息，无需反复来回调整
- 🗂️ **双格式输出** — 同时生成 HTML（在线预览）和 DOCX（Word 可编辑）
- 🔍 **ATS 友好** — 关键词对齐目标岗位 JD，通过招聘系统筛选
- 🧠 **岗位定制，不套模板** — 后端工程师、产品经理、设计师、运营、市场……每种岗位有独立的技能维度逻辑
- 💡 **高阶顾问建议** — 简历生成后自动附上 3-5 条精准求职建议，站在比你高 1-2 级的视角帮你审视
- 🚫 **无生成水印** — 输出简历完全归属于你

---

## 🚀 快速开始

将 `resume-builder-skill.md` 的内容作为 System Prompt 或 Skill 注入你使用的 Agent / LLM，然后直接告诉它：

```
帮我生成一份简历，我要投产品经理岗位，我的工作主要是……
```

或者：

```
帮我改进这份简历（附上现有简历内容）
```

Skill 会引导你完成信息收集，然后一次性生成专业简历。

---

## 🔄 工作流程

```
用户提供背景信息
       ↓
  信息收集阶段
  · 个人基本信息
  · 教育背景
  · 工作经历 + 量化成就
  · 岗位定制技能
  · 项目经验（可选）
       ↓
  简历生成阶段
  · HTML 版本（支持在线预览 + 打印为 PDF）
  · DOCX 版本（Word / WPS 可直接编辑）
       ↓
  高阶顾问建议
  · 3-5 条精准求职建议
  · 面试高频考点预判
  · 个人亮点提炼
```

## 📋 输出示例

### 工作经历分点格式

每条工作描述采用 **"加粗概括 + 冒号 + 详细描述"** 格式，让面试官 5 秒内抓住重点：

```
• 系统架构设计：主导设计高并发分布式支付系统，日均处理交易 100 万+，可用性 99.99%
• 性能优化：优化核心查询链路，平均响应时间从 500ms 降至 50ms，提升 10 倍
• 团队管理：带领 5 人小组完成 Q3 季度目标，提前 2 周交付
```

### 高阶顾问建议示例

```
💡 高阶顾问建议（以资深产品总监视角）

1. 补充商业化认知：你的经历集中在效率工具侧，但目标岗位是「商业化产品经理」，
   建议补充你对营收指标、定价策略的理解
2. 亮点提炼 — 变革推动力：从纸质到线上不只是产品设计，更是组织变革故事，
   可在面试中重点强调推动阻力和落地策略
3. 面试高频考点：B 端商业化岗大概率会问「如何衡量产品商业价值」，
   建议用 ROI 框架提前准备
```
